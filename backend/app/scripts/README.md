# Скрипты для заполнения базы данных

- Спорт и отдых (7 подкатегорий)

**Всего:** 2 основные категории, 24 категории уровня 2, 160+ категорий уровня 3

### Способы заполнения

#### Способ 1: Запуск Python скрипта (требуется подключение к БД)

```bash
cd backend
PYTHONPATH=/path/to/backend python app/scripts/seed_categories.py
```

#### Способ 2: Генерация и запуск SQL (рекомендуется)

1. Сгенерировать SQL файл:
```bash
cd backend/app/scripts
python generate_categories_sql.py
```

2. Запустить SQL в PostgreSQL:
```bash
psql -U postgres -d bazarlar_online -f categories_seed.sql
```

Или через pgAdmin:
- Откройте Query Tool
- Загрузите файл `categories_seed.sql`
- Выполните скрипт

### Файлы

- `seed_categories.py` - Python скрипт для автоматического заполнения (требует подключения к БД)
- `generate_categories_sql.py` - Генератор SQL скрипта
- `categories_seed.sql` - Готовый SQL скрипт (создаётся автоматически)

### Примечания

- Категории создаются с ID начиная с 1000, чтобы не конфликтовать с существующими
- Все категории активны (`is_active = true`)
- Для категорий уровня 1 и 2 указаны иконки Material Icons
- Slug генерируется автоматически из названия
- Поддерживается полная иерархия (parent-child relationships)
